What is Vue JS?

  • A Front-end JavaScript Framework
  • Created by Evan You
  • Can be used as a library or for building full JavaScript applications
  • Small bundle size, and excellent performance
  • Second most starred project on GitHub
  • Not directly sponsored by any major company
  • 5 years old - youngest, by less than a year
  • Evan You worked for Google as part of Angular team
  • Wanted to take what he liked about Angular and make it faster
  • Virtual DOM
  • Part of Laravel core
  • Supported by Patreon

Why should I care about Vue?

  • Easy learning curve
  • Great documentation
  • Massive growth in the industry
  • Helpful community
